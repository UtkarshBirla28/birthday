## Image Formats Supported
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## How to Replace

1. Delete the placeholder images (if any)
2. Add your own images with the exact names listed above
3. That's it! The website will automatically use your images

## Tips

✨ Use high-quality images
✨ Crop images to recommended sizes for best results
✨ Use consistent style/theme
✨ Test locally with `npm run dev` before deploying

## Fallback

If any image is missing, the website will automatically use placeholder images from picsum.photos, so your site will still work!
